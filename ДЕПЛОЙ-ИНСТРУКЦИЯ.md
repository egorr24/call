## 🚀 ДЕПЛОЙ НА RAILWAY - 3 ПРОСТЫХ ШАГА:

### 1️⃣ ЗАГРУЗИТЕ ПРОЕКТ
1. Зайдите на [railway.app](https://railway.app)
2. Нажмите "New Project"
3. Выберите "Deploy from GitHub repo" или "Upload files"
4. Загрузите всю папку `flux-deploy`

### 2️⃣ ДОБАВЬТЕ БАЗУ ДАННЫХ
1. В проекте нажмите "New Service"
2. Выберите "PostgreSQL"
3. Дождитесь создания (Railway автоматически создаст DATABASE_URL)

### 3️⃣ НАСТРОЙТЕ ПЕРЕМЕННЫЕ
В настройках основного сервиса добавьте:
```
NODE_ENV=production
JWT_SECRET=flux_super_secret_key_2024_make_it_very_long_and_secure_for_production_use
```

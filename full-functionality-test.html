🚀 Тест полного функционала мессенджера

✅ Что теперь работает:

Чаты: Можно выбирать чаты, они становятся активными, загружаются сообщения
Отправка сообщений: Сообщения отправляются через WebSocket или HTTP API, показываются в реальном времени
Поиск пользователей: Можно искать пользователей, создавать новые чаты, добавлять в контакты
Поиск чатов: Можно искать среди существующих чатов по имени или последнему сообщению
Настройки профиля: Можно изменять имя пользователя, пароль, сохранять изменения
Темы оформления: Можно выбирать и применять разные темы
Загрузка файлов: Можно прикреплять и отправлять файлы (до 10MB)
Модальные окна: Все окна открываются/закрываются, работают кнопки назад и закрытия
Уведомления: Показываются уведомления о всех действиях пользователя
Реальное время: Новые сообщения приходят через WebSocket, обновляется список чатов

🔧 Основные функции:

selectChat(chat): Выбор чата, загрузка сообщений, обновление интерфейса
sendMessage(): Отправка сообщений с временным отображением и подтверждением
searchUsers(query): Поиск пользователей с debounce, создание новых чатов
loadChatMessages(chatId): Загрузка истории сообщений чата
handleFileUpload(): Загрузка и отправка файлов с проверкой размера
saveProfileSettings(): Сохранение изменений профиля пользователя
filterChats(query): Фильтрация чатов по поисковому запросу

🎯 Как использовать:

  1. Войдите в аккаунт - регистрация и авторизация работают
  2. Выберите чат - кликните на чат в списке слева
  3. Отправьте сообщение - введите текст и нажмите Enter или кнопку отправки
  4. Найдите пользователей - кнопка "👥" в верхней панели
  5. Настройте профиль - кнопка "⚙️" → "Редактировать профиль"
  6. Прикрепите файл - кнопка "📎" в поле ввода сообщения
  7. Ищите чаты - поле поиска над списком чатов

📡 API Endpoints используются:

GET /api/me - проверка авторизации
GET /api/chats - загрузка списка чатов
GET /api/messages/:chatId - загрузка сообщений чата
POST /api/messages - отправка сообщения
GET /api/users/search - поиск пользователей
POST /api/chats/create - создание нового чата
PUT /api/profile/update - обновление профиля
POST /api/upload - загрузка файлов

## Как тестировать

### 1. Базовое тестирование
1. Откройте `http://localhost:8080/test-functionality.html`
2. Протестируйте регистрацию и вход
3. Проверьте API endpoints

### 2. Основное приложение
1. Откройте `http://localhost:8080`
2. Зарегистрируйтесь или войдите
3. Проверьте:
   - Отображение профиля пользователя
   - Загрузку чатов
   - Открытие модальных окон (статус, пользователи, настройки)
   - Создание чата с пользователем
   - Отправку сообщений

### 3. Модальные окна
- **Статус**: Должно открываться и позволять выбор статуса
- **Пользователи**: Должно показывать список пользователей с кнопкой "Написать"
- **Настройки**: Должно открываться с вкладками (профиль, темы, приватность)

### 4. Функциональность чата
- Выбор чата из списка
- Отправка текстовых сообщений
- Отображение сообщений в реальном времени

## Логи для отладки

Все действия логируются в консоль браузера с префиксом `🔥`. Откройте Developer Tools (F12) для просмотра логов.

## Известные ограничения

1. Файлы и голосовые сообщения требуют дополнительной настройки
2. Видеозвонки требуют WebRTC настройки
3. Некоторые новые функции (игры, опросы) требуют дополнительной реализации

## 🛠 Технологии

### Backend
- **Node.js** - Серверная платформа
- **Express.js** - Веб-фреймворк
- **Socket.IO** - Реальное время
- **JWT** - Аутентификация
- **Multer** - Загрузка файлов
- **bcryptjs** - Хеширование паролей

### Frontend
- **Vanilla JavaScript** - Чистый JS без фреймворков
- **HTML5** - Современная разметка
- **CSS3** - Cyberpunk стили и анимации
- **Socket.IO Client** - Клиент для реального времени

### База данных
- **PostgreSQL** (опционально)
- **Sequelize** - ORM
- **Memory Storage** - Fallback режим

## 🔧 API Endpoints

### Аутентификация
- `POST /api/register` - Регистрация
- `POST /api/login` - Вход
- `GET /api/me` - Получить профиль

### Чаты
- `GET /api/chats` - Список чатов
- `POST /api/chats/create` - Создать чат
- `GET /api/chats/:id/messages` - Сообщения чата

### Сообщения
- `POST /api/messages` - Отправить сообщение
- `POST /api/upload` - Загрузить файл

### Пользователи
- `GET /api/users` - Список пользователей
- `PUT /api/profile` - Обновить профиль

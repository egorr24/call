# Видеозвонки для Railway

Полноценное приложение для видеозвонков с WebSocket сервером.

## Деплой на Railway

1. Создай аккаунт на [Railway.app](https://railway.app)
2. Подключи GitHub репозиторий или загрузи файлы
3. Railway автоматически определит Node.js проект
4. Деплой произойдет автоматически

## Локальная разработка

```bash
npm install
npm run dev
```

## Возможности

- ✅ Реальные видеозвонки через WebRTC
- ✅ WebSocket сигнальный сервер
- ✅ Создание комнат по ссылкам
- ✅ Управление видео/аудио
- ✅ Демонстрация экрана
- ✅ Автоматическая очистка старых комнат

## Технологии

- **Backend**: Node.js + Express + Socket.io
- **Frontend**: Vanilla JavaScript + WebRTC
- **Деплой**: Railway (бесплатно)

# 🚀 Flux Messenger - Полнофункциональный мессенджер

1. **Голосовые сообщения** - запись и воспроизведение ✅
2. **Реакции на сообщения** - современно и удобно ✅
3. **Групповые чаты** - расширяет функционал ✅
4. **Темы оформления** - персонализация ✅
5. **Стикеры и GIF** - встроенная библиотека ✅

7. **Опросы и голосования** ✅

9. **Статусы и истории** (как в WhatsApp/Instagram) ✅
10. **Ответы на сообщения** (reply/quote) ✅
11. **Пересылка сообщений** ✅
12. **Видеозвонки и аудиозвонки** ✅

## 🛠 Технические особенности

### Backend
- **Node.js + Express** - серверная часть
- **Socket.IO** - реальное время
- **PostgreSQL** - основная база данных
- **JWT авторизация** - безопасность
- **Multer** - загрузка файлов
- **Fallback режим** - работа без БД

### Frontend
- **Vanilla JavaScript** - без фреймворков
- **CSS3 + Glassmorphism** - современный дизайн
- **WebRTC** - видеозвонки
- **Canvas API** - рисование
- **MediaRecorder API** - голосовые сообщения
- **Responsive Design** - адаптивность

### Безопасность
- **Хеширование паролей** - bcrypt
- **Валидация файлов** - проверка типов и размеров
- **Санитизация данных** - защита от XSS
- **CORS настройки** - безопасные запросы

## 📱 Мобильная поддержка

- **Адаптивный интерфейс** - оптимизация для всех экранов

## 🚀 Установка и запуск

1. **Клонирование репозитория**
```bash
git clone <repository-url>
cd messenger-app
```

2. **Установка зависимостей**
```bash
npm install
```

3. **Настройка базы данных (опционально)**
```bash
# Установите PostgreSQL и создайте базу данных
# Добавьте DATABASE_URL в переменные окружения
export DATABASE_URL="postgresql://username:password@localhost:5432/messenger"
```

4. **Запуск сервера**
```bash
npm start
# или для разработки
npm run dev
```

5. **Открытие в браузере**
```
http://localhost:3000
```

## 🚀 Деплой на Railway

### 1. Создание проекта
1. Зарегистрируйся на [Railway.app](https://railway.app)
2. Создай новый проект
3. Загрузи файлы из папки `messenger-app/`

### 2. Добавление PostgreSQL базы данных
1. В проекте Railway нажми **"+ New"**
2. Выбери **"Database"** → **"PostgreSQL"**
3. Railway автоматически создаст базу и установит переменную `DATABASE_URL`

### 3. Настройка переменных окружения (опционально)
```bash
JWT_SECRET=your-super-secret-jwt-key-here
NODE_ENV=production
```

### 4. Деплой
Railway автоматически:
- Установит зависимости (`npm install`)
- Создаст таблицы в PostgreSQL
- Запустит сервер (`npm start`)
- Даст тебе URL типа `https://flux-messenger.railway.app`

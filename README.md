# Flux Messenger

Современный мессенджер нового поколения с поддержкой голосовых сообщений, видеозвонков, игр и многого другого.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd messenger-app
npm install
```

### 2. Запуск сервера
```bash
npm start
```

Сервер запустится на порту 8080 (или PORT из переменных окружения).

### 3. Открыть приложение
- Основное приложение: `http://localhost:8080`
- Тестирование функций: `http://localhost:8080/test-functionality.html`

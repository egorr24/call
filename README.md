# 🔥 Flux Messenger

Современный мессенджер нового поколения с киберпанк дизайном.

## ✨ Возможности

- 💬 **Мгновенные сообщения** - Отправка текстовых сообщений в реальном времени
- 🎤 **Голосовые сообщения** - Запись и отправка аудио
- 📁 **Отправка файлов** - Поддержка изображений, видео, документов
- 😊 **Стикеры и эмодзи** - Выражение эмоций
- 🎮 **Игры в чате** - Крестики-нолики, викторины, угадай слово
- 📞 **Видео/аудио звонки** - WebRTC звонки между пользователями
- 🎨 **Совместное рисование** - Рисование в чате
- 🌙 **Темная тема** - Киберпанк дизайн с неоновыми акцентами
- 📱 **Адаптивный дизайн** - Работает на всех устройствах

## 🚀 Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Запуск в продакшене
```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## 🔧 Конфигурация

### Переменные окружения

Создайте файл `.env` на основе `.env.example`:

```bash
# Порт сервера
PORT=3000

# JWT секрет для токенов
JWT_SECRET=your-secret-key-here

# База данных PostgreSQL (опционально)
DATABASE_URL=postgresql://username:password@localhost:5432/flux_messenger

# Режим работы
NODE_ENV=development
```

### База данных

Приложение может работать в двух режимах:
- **Memory mode** (по умолчанию) - данные хранятся в памяти
- **PostgreSQL mode** - при наличии DATABASE_URL

## 📦 Деплой

### Railway
1. Подключите репозиторий к Railway
2. Установите переменные окружения
3. Деплой произойдет автоматически

### Heroku
1. Создайте приложение: `heroku create your-app-name`
2. Установите переменные: `heroku config:set JWT_SECRET=your-secret`
3. Деплой: `git push heroku main`

### Docker
```bash
# Сборка образа
docker build -t flux-messenger .

# Запуск контейнера
docker run -p 3000:3000 flux-messenger
```

## 🛠 Технологии

- **Backend**: Node.js, Express.js, Socket.IO
- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **База данных**: PostgreSQL (опционально)
- **Аутентификация**: JWT
- **Файлы**: Multer
- **WebRTC**: Видео/аудио звонки

## 🎨 Дизайн

Приложение использует киберпанк тему с:
- Темным фоном (#0a0a0f)
- Неоновыми акцентами (зеленый, розовый, синий)
- Анимированным фоном с сеткой
- Эффектами стекла и свечения
- Шрифтом JetBrains Mono
